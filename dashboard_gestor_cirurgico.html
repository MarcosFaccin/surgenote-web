Configure a conexão

Este painel ainda não está apontado para o seu projeto Supabase.

  1. Abra este arquivo num editor e localize o bloco CONFIG no topo do <script>.
  2. Preencha SUPABASE_URL e SUPABASE_ANON_KEY (Supabase → Settings → API).
  3. No Supabase, crie um usuário para o gestor em Authentication → Users.
  4. Rode o arquivo schema_gestao_cirurgica.sql no SQL Editor, salve e recarregue esta página.
